# hsi_UMAP
umap for geospatial applications

requirements: 
- conda env create -f umap39.yml

1. inputs:
   - attributed shapefile of vegetation ROIs
   - georeferenced multispectral imagery (geotiff, ENVI, etc.)
   - species lookup table (csv)
   - wavelength list
2. outputs:
   - 3D visualization
   - ROI statistics
